MTE-SLAM: Multi-Tier Feature Fusion for Efficient Neural Semantic SLAM

Danqi Lu, Changxin Huang, Zhuangzhuang Chen, Zhiliang Lin, Dachong Li, Yanbin Chang, Jianqiang Li* School of Artificial Intelligence, Shenzhen University

Paper Code

Overview

Existing semantic neural implicit SLAM systems often underutilize semantic cues. Most methods either inject semantics through simple coloring or perform shallow one-stage fusion, which frequently causes blurred object boundaries, fragmented small structures, and weaker local consistency under clutter or occlusion.

MTE-SLAM addresses this by jointly designing a global-to-local semantic fusion pipeline and an adaptive redundancy suppression mechanism. The global branch models scene-level context, the local branch enforces neighborhood continuity, and the suppressor removes low-value semantic features over time. This combination improves semantic detail quality while keeping mapping and optimization efficient for practical per-frame processing.

Method

MTE-SLAM couples coarse-to-fine neural rendering with semantic-aware feature optimization. The system jointly supervises RGB, depth, and semantic consistency, while injecting global context and local continuity constraints into semantic fusion.

MTFF: Multi-Tier Feature Fusion

MTFF performs hierarchical semantic aggregation in two stages. The global context branch models long-range scene dependencies to stabilize category-level consistency, while the local continuity branch strengthens neighborhood-level boundary alignment. Their collaborative fusion improves small-object integrity, edge sharpness, and regional coherence under occlusion and fast viewpoint changes.

FRS: Feature Redundancy Suppressor

FRS adaptively suppresses low-value semantic channels based on feature importance and temporal variation. Instead of uniformly keeping all activations, it retains only informative components for mapping updates, reducing redundant computation and memory cost. This keeps semantic expressiveness while significantly improving frame-wise efficiency.
